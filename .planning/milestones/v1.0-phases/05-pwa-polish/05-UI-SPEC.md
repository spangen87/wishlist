## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding (`gap-1`, `p-1`) |
| sm | 8px | Compact element spacing (`gap-2`, `p-2`) |
| md | 16px | Default element spacing, card padding (`p-4`, `gap-4`) |
| lg | 24px | Section padding (`py-6`, `gap-6`) |
| xl | 32px | Layout gaps (`py-8`) |
| 2xl | 48px | Major section breaks (`py-12`) |
| 3xl | 64px | Page-level spacing (used in EmptyState `py-12` large icon) |

Exceptions:
- Touch targets: minimum 44px height (`min-h-[44px]`) for all interactive elements — established in Phases 3–4, maintained in all new components
- Toast/banner: fixed position elements use `px-4 py-3` (12px + 16px)

## Color

Phase 5 updates `--color-accent` and `--color-accent-hover` in `globals.css`. All other tokens remain.

| Role | Token | Value | Usage |
|------|-------|-------|-------|
| Dominant (60%) | `--color-bg` | `#FFF9F5` | Page backgrounds, `<html>/<body>` surface — source: D-02, already in globals.css |
| Secondary (30%) | `--color-card` + `--color-border` | `#FFF0E8` / `#E5D5CC` | Card backgrounds, borders, image placeholders — source: D-02, already in globals.css |
| Accent (10%) | `--color-accent` | `#F9A87A` | Primary CTA buttons, edit/action text links, active purchase toggle — replaces #F97316 (source: D-01) |
| Accent hover | `--color-accent-hover` | `#F49060` | Hover state for accent buttons — replaces #EA6C0A (source: D-01) |
| Destructive | `--color-destructive` | `#DC2626` | Delete actions and error messages only — unchanged |
| Text | `--color-text` | `#171717` | Primary body text — unchanged |
| Muted | `--color-muted` | `#6B7280` | Secondary text, labels, metadata — unchanged |
| Offline banner bg | — | `#FFF0E8` | Offline sticky banner background (matches card surface, non-alarming) |
| Update toast bg | — | `#FFF0E8` | SW update toast background (same warm card surface) |

Accent `#F9A87A` rationale: warmer than #F97316 (vivid orange), stays within peach/rose range, harmonizes with the peachy #FFF9F5 background. WCAG AA contrast against white: 2.9:1 — use white text only on filled buttons (large text at 16px semibold passes; reinforce with border if needed for smaller contexts). Source: D-01, Claude's discretion for exact hex.

Accent reserved for:
1. Primary CTA button fill (e.g., "Lägg till önskemål", "Spara önskemål", "Uppdatera nu")
2. Edit/action text link color (e.g., "Redigera" link on WishItemCard)
3. Active/filled state of "Markera som köpt" toggle button (ViewerWishItemCard)
4. Nothing else — muted text links and secondary actions use `text-[#6B7280]` or `text-[#171717]`

Offline/update UI must NOT use red or warning orange. Use warm neutrals (`#FFF0E8` bg, `#171717` text, `#6B7280` secondary text).

## Component Inventory

New components introduced in Phase 5:

### OfflineBanner
- Sticky bar at top of page (below any nav), `position: sticky top-0 z-50`
- Background: `#FFF0E8`, border-bottom: `1px solid #E5D5CC`
- Text: "Du ar offline — data kan vara foralдrad" (Swedish, 14px, weight 400, color `#171717`)
- Height: 44px minimum to meet touch target baseline
- Visible state: `window.addEventListener('offline')` triggers show; `'online'` triggers hide
- Animation: fade-in/fade-out with `transition-opacity duration-300`
- Placement: inside root layout, renders on all pages; visibility controlled by hook

### UpdateToast
- Fixed bottom-center bar: `position: fixed bottom-4 left-1/2 -translate-x-1/2 z-50`
- Background: `#FFF0E8`, border: `1px solid #E5D5CC`, `rounded-2xl shadow-md`
- Text: "Ny version tillganglig" (14px, weight 400, `#171717`)
- CTA button: "Uppdatera nu" — accent fill (`#F9A87A`), white text, `rounded-xl px-4 py-2 min-h-[44px] font-semibold text-sm`
- Non-modal, non-blocking, does not auto-dismiss (source: D-15)
- Animation: slide-up from bottom with `transition-transform duration-300`
- Placement: inside root layout alongside AuthProvider

### OfflineFallback page (`app/offline/page.tsx`)
- Full-page centered layout with `bg-[#FFF9F5]`
- Large emoji: "📶" or "⭐" at 64px (aria-hidden)
- Heading: "Du ar offline" — 28px semibold `#171717`
- Body: "Kontrollera din internetanslutning och forsok igen." — 16px regular `#171717`
- No primary CTA (no network action possible)

### PWA Icons (generated, not a React component)
- Output: `public/icons/icon-192.png` and `public/icons/icon-512.png`
- Design: `#FFF9F5` background + centered gift emoji "🎁" rendered in system font
- Generated via Node.js script using `sharp` or `canvas` npm package (source: D-09)

### Web App Manifest (`public/manifest.json` or auto-generated by @ducanh2912/next-pwa)
- `name`: "Min onskelista" (source: D-06)
- `short_name`: "Onskelista" (source: D-06)
- `theme_color`: `#FFF9F5` (source: D-07)
- `background_color`: `#FFF9F5` (source: D-07)
- `display`: `standalone` (source: D-08)
- `icons`: 192x192 and 512x512 from `public/icons/`

## Card Design Contract

All cards across all pages use (source: D-04):

```
bg-[#FFF0E8] border border-[#E5D5CC] rounded-2xl shadow-sm hover:shadow-md transition-shadow p-4
```

This is already the established pattern in WishItemCard, ViewerWishItemCard, WishlistDashboardCard. Phase 5 enforces consistency — no card should deviate from `rounded-2xl shadow-sm`.

OfflineBanner and UpdateToast do NOT use card styling — they are system-level overlays using a lighter treatment described in Component Inventory above.

## Interaction States

### OfflineBanner
| State | Visual |
|-------|--------|
| Online (default) | Hidden (`opacity-0 h-0 overflow-hidden`) |
| Offline | Visible (`opacity-100 h-11`) — fade transition 300ms |
| Back online | Auto-hide with fade-out 300ms |

### UpdateToast
| State | Visual |
|-------|--------|
| No waiting SW | Hidden (not rendered) |
| New SW waiting | Slides up from bottom 300ms |
| "Uppdatera nu" clicked | Button shows loading state (disabled + opacity-50), page reloads |

### Primary CTA Buttons (accent color update applies everywhere)
| State | Visual |
|-------|--------|
| Default | `bg-[#F9A87A] text-white` |
| Hover | `bg-[#F49060] text-white` |
| Disabled | `opacity-50 cursor-not-allowed` |
| Loading/saving | `opacity-50 cursor-not-allowed` + button text unchanged |
